## 🚀 스터디 진행 방식
- 매주 금요일 2시에 스터디 진행
- 스터디는 "가상면접 사례로 배우는 대규모 아키텍처 시스템 설계 기초" 책을 바탕으로 진행
- 구성원 한명당 30분씩 돌아가며 책 내용(혹은 다루고 싶은 주제)에 대해 면접
- 클로바 노트로 면접 기록
- 구성원 별 면접 종료 후 면접 질의응답 디스커션 남겨 복기
